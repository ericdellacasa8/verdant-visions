# Verdant & Bloom - Gardening Business Website

A polished, single-page brochure website for a premium gardening business.

## Features

- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Modern UI**: Clean typography, generous whitespace, and high-quality imagery.
- **Plant Gallery**: Grid layout showcasing featured plants with hover effects.
- **Services Section**: Clear overview of business offerings.
- **Contact Form**: Stylish layout for customer inquiries.
- **Tailwind CSS**: Built with utility-first CSS for rapid and maintainable styling.

## Technology

- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: Modern styling framework.
- **Vanilla JavaScript**: Lightweight interactions (mobile menu, scroll animations).

## Setup

No build step required. Simply open `index.html` in your browser or serve it with a local static server.

```bash
# Using python simple server
python3 -m http.server 8080
```

Access at `http://localhost:8080`
